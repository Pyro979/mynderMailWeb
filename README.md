# mynderMail Documentation

This repository contains the public documentation and legal pages for mynderMail, hosted on GitHub Pages.

## Pages

- **Privacy Policy** - `privacy-policy.html`
- **Terms of Service** - `terms-of-service.html`
- **Help & FAQ** - `help.html`

## GitHub Pages Setup

This repository is configured to use GitHub Pages. The site will be available at:
`https://pyro979.github.io/mynderMailWeb/`

## Local Development

To preview the site locally:

1. Install Jekyll (if using Jekyll features):
   ```bash
   gem install bundler jekyll
   ```

2. Serve locally:
   ```bash
   jekyll serve
   ```

3. Open `http://localhost:4000/mynderMailWeb/` in your browser

Or simply open the HTML files directly in a browser for basic preview.

## Updating Content

1. Edit the HTML files directly
2. Commit and push changes
3. GitHub Pages will automatically rebuild and deploy

## Notes

- All pages use inline CSS for simplicity
- Pages are mobile-responsive
- Links are relative for GitHub Pages compatibility
