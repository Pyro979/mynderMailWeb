Help & FAQ

1. Getting Started

How to Send a Message

  • Type your message in the text area
  • The first line automatically becomes the email subject
  • Tap "Send" to send immediately, or use a quick action button to categorize

Quick Actions

  • Quick action buttons let you categorize messages (Todo, Idea, Reminder, etc.)
  • Customize quick actions in Settings
  • On small screens, use the dropdown menu next to the send button

2. Features

Draft Saving

  • Your message is automatically saved as you type
  • Drafts persist across app restarts
  • Draft is cleared when message is successfully sent

Offline Support

  • Messages are queued when offline
  • They're sent automatically when you're back online
  • Check the history button badge to see unsent message count

Pinned Notification

  • Enable in Settings for quick access from anywhere
  • Tap the notification to compose a new message
  • Works even when the app is closed

Quick Settings Tile

  • Add mynderMail to your Quick Settings panel for fastest access
  • To add: Long-press any Quick Settings tile → Edit tiles → Add mynderMail

Share Integration

  • Share text from other apps to mynderMail
  • Configure behavior in Settings (auto-send or open compose screen)

3. Settings

Recipient Email

  • Set your default recipient email address
  • Quick actions can modify this (e.g., +todo modifier)

History Retention

  • Choose how many sent messages to keep
  • Unsent messages (queued/failed) are always kept
  • Reducing retention will delete older sent messages

Theme

  • Choose Light, Dark, or System default theme
  • Theme applies immediately

Close App on Send

  • Automatically close the app after sending for faster workflow
  • Disable if you want to stay in the app

Crash Reporting

  • Help improve mynderMail by sending crash reports
  • No personal information is included in crash reports
  • Can be enabled/disabled in Settings

4. Troubleshooting

Messages Not Sending

  • Check that you're signed in with Google (Settings → Gmail Account)
  • Verify your recipient email is set (Settings → Recipient Email)
  • Check your internet connection
  • Failed messages can be retried from History

Notification Not Showing

  • Enable "Pinned Notification" in Settings
  • Grant notification permission when prompted (Android 13+)
  • Check that notification channel isn't disabled in system settings

Can't Add Quick Settings Tile

  • Quick Settings tiles must be added manually
  • Long-press any tile in Quick Settings → Edit tiles → Add mynderMail
  • If you don't see mynderMail, make sure the app is installed
